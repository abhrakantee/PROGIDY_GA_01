# Task 01: Text Generation with GPT-2

## Description
This project involves fine-tuning the GPT-2 transformer model on a custom dataset to generate coherent and contextually relevant text.

## Tools Used
* **Model:** GPT-2 (via Hugging Face)
* **Environment:** Google Colab (GPU: T4)
* **Language:** Python

## How to Run
1. Upload the `data.txt` file.
2. Run the provided Jupyter Notebook (`.ipynb`).
3. The model will train and output generated text at the end.
